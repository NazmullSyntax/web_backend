# Notes Management System - Frontend

A beautiful, fully functional Notes Management System built with HTML, CSS, and JavaScript. This frontend application simulates backend functionality using localStorage.

## Features

### ✅ Authentication & Authorization
- User registration and login
- JWT token simulation
- Role-based access (User/Admin)
- Secure password handling

### ✅ CRUD Operations
- Create, Read, Update, Delete notes
- Update note status (active/archived/deleted)
- Delete all notes functionality
- Search notes with advanced filters

### ✅ File Management
- Upload images, PDFs, documents
- 5MB file size limit
- File preview and download
- Organized file storage

### ✅ Advanced Features
- Search with partial match
- Category-based filtering
- Priority levels (low/medium/high)
- Tag system for organization
- Responsive design for all devices

### ✅ Admin Panel
- User management
- System statistics
- Data export/import
- Clear all data functionality
